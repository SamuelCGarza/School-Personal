In Chapter 3, Exercise 6 you used counter objects to track the number of instructions performed in sort algorithms.

In this exercise you will add a counter to the maze program of Exercise 9 to count the number of cells visited while the program is solving the maze.

In the Counter class of the counter.py file, complete the following:

Complete the implementation of the Counter class.
Define the class variable instances to 0
__init__, sets up the counter
reset(), sets the counter to 0
increment(), adds amount to the counter.
decrement(), subtracts amount from the counter.
getValue(), returns the counter's value.
__str__, returns the string representation of the counter.
__eq__, returns True if self equals other or False otherwise.
To test your program run the main() method in the maze.py file.
