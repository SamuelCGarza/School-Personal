In this exercise you will find the files hashtable.py and profiler.py as the Chapter 11 case study files.

Complete the profiler for hash tables begun in the Chapter 11 case study files. This includes creating the following methods in the hashtable.py file:

__len__
__str__
getLoadFactor()
getHomeIndex()
getActualIndex()
getProbeCount()
and the following methods in the Profiler class of the profiler.py file:

getCollisions()
getProbeCount()
